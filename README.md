## Development

1. Download Dagger: `make bin/dagger`
1. Write some code (and tests)
1. Build the app: `make build`
1. Run tests: `make test`
1. Run lint: `make lint`
1. Commit and push

**Pro tip:** Use `dagger call ci` to run the entire CI pipeline locally.
